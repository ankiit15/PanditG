# 🕉️ PanditG – AI-powered Palmistry & Horoscope App

**PanditG** is a next-generation mobile/web app that blends ancient Vedic wisdom with modern artificial intelligence to deliver personalized palmistry, astrological predictions, and spiritual guidance. Whether you believe in the stars or just love a good personality insight, PanditG adapts to you.

---

## 🚀 Features

- ✋ **AI-Based Palm Reading** – Computer vision model scans palm lines to detect personality and life trends.
- 🪐 **Horoscope Generator** – Generates daily/monthly horoscope based on birth details.
- 🧠 **Mode Selector** – Choose between:
  - **Vedic Mode** (traditional astrological advice)
  - **Scientific Mode** (psychological + behavior-based insights)
  - **Experimental Mode** (fun predictions with data visualizations)
- 💬 **PanditG ChatBot** – Ask anything! Career? Love? Life direction? PanditG responds with poetic or logical insights.
- 🌐 **Multilingual UI** – English, Hindi, Marathi (more coming)
- 📜 **Exportable Reports** – Personalized destiny snapshot in PDF

## 🧪 Tech Stack

| Layer | Tools Used |
|-------|------------|
| **Frontend** | ReactJS / Flutter, TailwindCSS, Figma (UI Design) |
| **Backend** | Node.js / Flask + Express, REST APIs |
| **AI/ML** | TensorFlow / OpenCV (Palm Scanner), NLP (ChatBot), Astrology API |
| **Database** | MongoDB / Firebase |
| **Cloud/Deployment** | Render / Vercel / Firebase Hosting |
| **Version Control** | Git + GitHub |
